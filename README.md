# College Attendance System - Mobile App

React Native mobile application for GPS-based attendance tracking with video verification.

## Features
- Student attendance marking with GPS + 5-sec video + face snapshot
- Faculty session management
- Real-time attendance tracking
- Admin dashboard

## Tech Stack
- React Native + Expo SDK 52
- Backend: Node.js + SQLite
- Authentication: JWT

## Setup
```bash
npm install
npx expo start
```

## Build APK
```bash
eas build --platform android --profile preview
```
